# Create Analysis

Source: <https://doc.alvao.com/en/25/modules/alvao-am-custom-apps/applications/templates/#create-analysis>

When a new ticket is created in the Program modifications service, a linked ticket is also created in the Program modifications/Designs and analyzes service to create a design for the created ticket.

To run the example, a custom field createAnalysis of type bool is assumed to be added to the tHdTicketCust table.
