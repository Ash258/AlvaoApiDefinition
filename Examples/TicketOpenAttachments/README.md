# Ticket Open Attachments

Source: <https://doc.alvao.com/en/25/modules/alvao-am-custom-apps/applications/templates/#ticket-open-attachments>

To the Ticket page, adds the Open attachments command for opening an attachment link saved in Attachments custom field of the ticket.

To execute the example, it requires adding custom field named Attachments of type nvarchar to the tHdTicketCust table and naming the column in the same way.
