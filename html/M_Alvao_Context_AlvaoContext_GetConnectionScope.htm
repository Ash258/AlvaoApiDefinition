IConnectionScope
Connection scope based on the context provider DbContextProvider.
Example
The GetById method is used to retrieve person's data from the database and shows the use of the GetConnectionScope method.
C#
Copy 
public tPerson GetById(int personId)
{
    using (var scope = AlvaoContext.GetConnectionScope())
    {
        return scope.Connection.QueryFirst<tPerson>(
            @"SELECT * FROM tPerson WHERE iPersonId = @personId",
            new { personId },
            scope.Transaction);
    }
}
