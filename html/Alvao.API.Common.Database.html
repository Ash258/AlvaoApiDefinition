Parameters

Record ID.

tableName string

Column name, case sensitive.

newValue string

Textual value that must be convertible to the target column type. Null is allowed, if specified then null is written to the column.

Data type of the newValue parameter. Usable only for integer custom columns with value list when you need to set the text value instead of its Id.
