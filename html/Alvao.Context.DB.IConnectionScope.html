Examples

The example contains the code used to search the person's ID in the database. It uses the GetConnectionScope() method to obtain the database connection scope. Scope properties LegacyConnection and LegacyTransaction are used in the parameters of SqlCommand constructor.

public static int? GetPersonIdByAzureId(string azureObjectId)
{
    using (var scope = AlvaoContext.GetConnectionScope())
    {
        using (var cmd = new SqlCommand(
            "SELECT iPersonId FROM tPerson WHERE AzureAdObjectId = @azureObjectId",
			scope.LegacyConnection,
			scope.LegacyTransaction))
        {
            cmd.Parameters.AddWithValue("@azureObjectId", azureObjectId);
            return cmd.ExecuteScalar() as int?;
        }
	}
}
