Parameters

tableName string

Custom table name, use consts from Alvao.API.Common.Model.CustomTables. Supported tables are CustomTables.tHdTicket, CustomTables.tHdSection, CustomTables.tPerson and CustomTables.tAccount

columnName string

Custom column name

entityId int

Custom table entity identificator (eg. tHdTicket.iHdTicketId for CustomTables.tHdTicket, tPerson.iPersonId for CustomTables.tPerson).

language CultureInfo

Column value language (for localized values)

timeZone string

Column value time zone (for DateTime values)

returnPersonName bool

If true, for custom column type user returns it's name, otherwise returns it's id. Default is true.

returnBitAsNumber bool

If true, for custom column type bit returns "1"/"0", otherwise returns localized values "Yes"/"No". Default is true.

formatDouble bool

If true, for custom column type float returns localized value in the preferred language. Default is false.
