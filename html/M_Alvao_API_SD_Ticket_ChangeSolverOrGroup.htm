public static int ChangeSolverOrGroup(
    int ticketId,
    int? solverId = -1,
    int? solverGroupId = -1,
    HtmlTextModel message = null,
    CostModel cost = null,
    int? authorId = null,

Parameters

ticketId  Int32
Request ID (tHdTicket.iHdTicketId)
solverId  NullableInt32  (Optional)
Person ID of new solver. Null means that ticket is returned to main solver. If set to -1 then solver stays unchanged.
solverGroupId  NullableInt32  (Optional)
Role ID of new solver group. Null means that ticket is returned to main solver. If set to -1 then solver group stays unchanged.
message  HtmlTextModel  (Optional)
Message for solver or solver group
cost  CostModel  (Optional)
Costs
authorId  NullableInt32  (Optional)
Person ID of change author. In case of null, current user will be used.
flags  TicketChangeSolverFlags  (Optional)
Flags for special cases.

Return Value

Int32
0 if no act was logged otherwise act ID (tAct.iActId) with log of solver or solver group change.
Exceptions
Exception Condition
ArgumentException Throws when request with ticketId does not exist.
